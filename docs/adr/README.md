# Architecture Decision Records

This folder holds short documents describing the **why** behind significant technical decisions on Acca. Each ADR is numbered, dated, and immutable once accepted — if a later decision overrides it, write a new ADR rather than editing the old one.

The format we use: **Context → Decision → Consequences**, kept to one page.
